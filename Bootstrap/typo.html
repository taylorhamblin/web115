These elements provide meaning "through the color":

"text-muted" = text is muted.

"text-primary" = text is important.

"text-success" = text indicates success.

"text-info" = text represents some information.

"text-warning" = text represents a warning.

"text-danger" = text represents danger.


These elements change the background color of the text:

"bg-primary" = text is important.

"bg-success" = text indicates success.

"bg-info" = text represents some information.

"bg-warning" = text represents a warning.

"bg-danger" = text represents danger.
